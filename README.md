# aktivitet-buddy

React Native-app for å finne aktivitetspartnere i Norge. Bygget med Expo, Supabase og Apple Maps.

## Kom i gang

### 1. Installer avhengigheter

```bash
npm install
```

### 2. Sett opp Supabase

1. Lag et prosjekt på [supabase.com](https://supabase.com)
2. Kopier `.env.example` til `.env` og fyll inn nøklene dine:
   ```bash
   cp .env.example .env
   ```
3. Kjør følgende SQL i **Supabase SQL Editor**:

```sql
-- Brukerprofiler (kobles til auth.users)
create table public.profiles (
  id uuid references auth.users on delete cascade primary key,
  username text unique,
  full_name text,
  avatar_url text,
  created_at timestamp with time zone default timezone('utc', now())
);

alter table public.profiles enable row level security;
create policy "Profiler er synlige for alle innloggede" on public.profiles
  for select using (auth.role() = 'authenticated');
create policy "Brukere kan oppdatere egen profil" on public.profiles
  for update using (auth.uid() = id);

-- Legg til profil automatisk ved ny bruker
create function public.handle_new_user()
returns trigger as $$
begin
  insert into public.profiles (id)
  values (new.id);
  return new;
end;
$$ language plpgsql security definer;

create trigger on_auth_user_created
  after insert on auth.users
  for each row execute procedure public.handle_new_user();

-- Aktiviteter
create table public.activities (
  id bigint generated by default as identity primary key,
  user_id uuid references auth.users on delete cascade not null,
  title text not null,
  type text not null,
  description text,
  latitude double precision not null,
  longitude double precision not null,
  created_at timestamp with time zone default timezone('utc', now())
);

alter table public.activities enable row level security;
create policy "Aktiviteter er synlige for alle innloggede" on public.activities
  for select using (auth.role() = 'authenticated');
create policy "Brukere kan lage egne aktiviteter" on public.activities
  for insert with check (auth.uid() = user_id);
create policy "Brukere kan slette egne aktiviteter" on public.activities
  for delete using (auth.uid() = user_id);
```

### 3. Start appen

```bash
npx expo start
```

Scan QR-koden med **Expo Go**-appen (iOS/Android).

## Mappestruktur

```
aktivitet-buddy/
├── app/
│   ├── (auth)/
│   │   ├── login.jsx       ← Innloggingsskjerm
│   │   └── register.jsx    ← Registreringsskjerm
│   └── (tabs)/
│       ├── map.jsx         ← Kart med aktivitetspins
│       ├── activities.jsx  ← Liste over aktiviteter
│       └── profile.jsx     ← Brukerprofil og utlogging
├── components/
│   ├── ActivityPin.jsx     ← Map-pin komponent
│   └── ActivityCard.jsx    ← Listevisning av aktivitet
├── lib/
│   └── supabase.js         ← Supabase-klient
├── hooks/
│   └── useAuth.js          ← Auth state hook
└── .env.example
```

## Teknologier

- [Expo](https://expo.dev) (SDK 55, blank JavaScript-template)
- [Supabase](https://supabase.com) — auth og database
- [React Navigation](https://reactnavigation.org) — navigasjon
- [react-native-maps](https://github.com/react-native-maps/react-native-maps) — Apple Maps (iOS)
- [expo-location](https://docs.expo.dev/versions/latest/sdk/location/) — GPS-posisjon
